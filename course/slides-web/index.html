AI 協作開發 · 實戰課程

從一個點子，
到一個能分享的網址。

學會把想法寫成「AI 照著做就能完成」的規格，再用 AI 編碼助手一步步做出一款 FHD 2D 精準平台跳躍網頁遊戲，最後免費部署到 GitHub Pages、拿到屬於你的遊戲網址。

10 堂課 錄播 7–9 小時 實作 12–24 小時 零基礎可跟
